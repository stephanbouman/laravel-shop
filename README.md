# Laravel Example Shop

This is my example shop. It is nothing more than an easy example for my students to show them of what Laravel can do for a developer. I've worked out some features in this implementation

- [Eloquent models](https://laravel.com/docs/8.x/eloquent)
- [Blade views](https://laravel.com/docs/8.x/blade#introduction)
- [Controllers](https://laravel.com/docs/8.x/controllers)
- [Routes](https://laravel.com/docs/8.x/routing)
- [Requests](https://laravel.com/docs/8.x/requests)
- [Sessions](https://laravel.com/docs/8.x/session)
- [Database migrations](https://laravel.com/docs/8.x/migrations)
- [Database seeders](https://laravel.com/docs/8.x/seeding)
- [Model factories](https://laravel.com/docs/8.x/database-testing#defining-model-factories)
- [Model relations](https://laravel.com/docs/8.x/eloquent-relationships)
- [Service container binding](https://laravel.com/docs/8.x/container)
